Musicpad

Musicpad Tutorial

Musicpad is a text notation for quickly sketching musical ideas and generating MIDI files. You do not need the whole language at once: start with notes, then add rhythm, tracks, drums, repeats, and macros.

1. Start with notes

Write note names separated by spaces. Notes are case-insensitive.

C E G F Eb C# C

Sharps can be written with # or +; flats with b or -.

2. Add duration

Add /N to make a note last 1/N. The default is 1/4.

C/4 E/4 G/2 F/8 Eb/8 C#/8 C/8

Musicpad remembers missing information. Once the duration is /8, following notes reuse it until changed:

C/4 E G/2 F/8 Eb C# C

3. Octaves

Put an octave number after a note, or use standalone / and \ to move up and down one octave.

C/4 E G/2 F/8 Eb C# C5/8
C/4 E G/2 F/8 Eb C# / C

The default octave is 4.

4. Repeat notes, hold notes, and pause

x repeats the previous note or chord.

C/8 x x x

= holds the previous note for one more duration unit. It can be attached directly.

C/8 E= G=== F Eb C# C

P is an explicit pause. - is a pause with the previous duration.

C/4 E G/2 P/4 F/8 Eb C# / C x x - x

Pattern characters do not need spaces. C/8 -x-xxx=x-x is valid.

5. Use o to set defaults without playing

o means “remember these note settings, but do not play now.” It is especially useful for rhythm patterns.

o/16 x-x-xx--

This sets the duration to sixteenth notes, then plays the pattern.

6. Tempo

Set the song tempo with tempoN. The number is quarter notes per minute.

tempo100
C/4 E G/2 P/4 F/8 Eb C# / C

7. Tracks and channels

Use | to start another independent track. Use chN to give each track a MIDI channel.

ch1 C E G | ch2 E G B

Change instruments with iN, using General MIDI program numbers.

ch1 i17 C E G | ch2 i25 E G B

You can jump back to a specific track with |0, |1, etc. This is useful when writing a song section by section.

ch1 i17 C E G | ch2 i25 E G B |0 F E C | A G E

Tracks are independent. If two tracks should stay aligned, you must give them matching total durations.

8. Drums

Drum shortcuts such as iBD, iSD, and iHH select a drum sound, switch to MIDI channel 10, and set the default note.

iBD o/16 xx-xxx--
| iSD o/16 --x---xx
| iHH o/16 -x-x-x-x

Each drum voice is easiest to read and control on its own track.

9. Repeats

*N repeats the previous token. Parentheses repeat a group.

iBD 8*8

iBD (8 4)*4

Because rhythm strings like xx-xxx-- are single compact tokens before parsing, this is valid:

iBD o/16 xx-xxx--*4
| iSD o/16 --x---xx*4
| iHH o/16 -x-x-x-x*4

10. Macros

Define reusable parts with m$name(...), then call them with $name. Macro names are case-sensitive.

m$riff(C/16 E G)
$riff $riff*3

Macros are most useful when they include track jumps.

m$drums(
|0 iBD o/16 xx-xxx--
|  iSD o/16 --x---xx
|  iHH o/16 -x-x-x-x
)

$drums*4

11. A typical song layout

One practical structure is: title/comment, global settings, track setup, macros, then song structure.

# My cool song

tempo70

# Instruments
|0 iHH o/16 | iSD o/16 | iBD o/16
|5 ch1 i01 o4/16 | ch2 i34 o2/16

# Macros
m$rhythm(
|0 xxxxxxxxxxxxxxxx
|  --x---x---x---x-
|  x---x---x---x---
)

m$bass1( |6 E*16 )
m$bass2( |6 A*16 )

m$piano1( |5 (P G# G# G# A G# F# G#)*2 )
m$piano2( |5 (P G# G# G# A B / C# E \)*2 )

# Song structure
$rhythm*4 ($bass1 $bass2 $piano1 $piano2)*2

12. Chords and strumming

For simple block chords, use brackets.

oC4 [0,4,7]
[Amin]
[g:-,0,2,2,2,0]
[g:Amaj]

Use strum to make chords less mechanical.

i25 o/16 strum10,300,80 [g:Amin]*16

13. Guitar mode

In guitar mode, bare numbers mean temporary semitone offsets from the default note. This makes tab-like writing possible.

tempo100
|0 ch1 i25 guiton | ch1 guiton | ch1 guiton
(
|0 oE5/16 ------5=7=====7=8=====8=2=====2=0=========0=====
|  oB4/16 ----5==---5====---5====---3====---1===1=====1===0=
|  oG4/16 --5====-----5==-----5==-----2==-----2=========2=0=
)*2

globalguiton turns guitar mode on by default for all tracks. Use guitoff to disable it on a track.

14. Tom mode

In tom mode, bare numbers choose toms: 1 is highest, 6 is lowest.

tomson o/16 4 3 5 1 1 5 6 6 4 5 4 2 1 1 1 6

15. More expressive commands

  • t+N, t-N, t0: transpose following notes.
  • N60, N+1, N-1: absolute or relative MIDI note numbers.
  • NT+1, NT-1: temporary relative notes.
  • Ra/b, R1: duration ratios for triplets and polyrhythms.
  • vN, velocityN: velocity.
  • 'x, ,x, stressN, softN: accents.
  • looseW,Q, velvarW,Q: humanize timing and velocity.
  • pitch+N, pitch-N, pitch0: pitch bends.

Musicpad Language Reference

This page is a practical reference for the Musicpad notation accepted by the JavaScript MIDI generator. Commands are case-insensitive unless stated otherwise. Whitespace separates commands; do not put whitespace inside one command such as tempo120 or C#5/8.

Basic lexical rules

  • Notes and commands are generally case-insensitive: c, C, Tempo120 work the same.
  • Exceptions: dEbUg, dEbUgMaX, and macro names are case-sensitive.
  • A comment starts with # when it follows whitespace and continues to the end of the line.
  • Whitespace is optional around pattern characters x, -, =, and parentheses used for repeats.
  • # after a note is sharp: C# is not a comment.

About Musicpad

Musicpad is a textual musical notepad: you write compact notation, then generate a MIDI file. The original project was designed as a fast way to capture musical ideas from any computer, using plain text instead of a graphical sequencer.

Original project

The original Musicpad was written by Loïc Prot as a CGI Perl script, largely inspired by Jens Johansson’s polymath.cgi. The archived pages describe Musicpad as an online musical notepad with support for notes, rhythms, tracks, macros, polyrhythms, random macros, MIDI controllers, pitch bends, guitar-oriented notation, drum shortcuts, and humanization commands.

What Musicpad is good at

  • Sketching melodies quickly with note names such as C E G.
  • Writing rhythm patterns compactly with x, -, and =.
  • Layering independent tracks with |.
  • Building songs from reusable macros and repeated sections.
  • Writing drum parts with short names like iBD, iSD, and iHH.
  • Representing guitar-tab-like material with guitar mode.

The “I” project

The archived Musicpad pages included a large “I” project: a Work in Progress transcription of Meshuggah’s 21-minute song I. It showed how Musicpad could be used for long, complex, heavily structured music, with many macros, dense rhythm patterns, and multiple tracks.

This JavaScript version

This application ports the Musicpad-to-MIDI functionality to JavaScript. It runs locally in the browser: you can load a Musicpad text file, edit it, save the source, and generate a MIDI file for download. Playback is intentionally left to your own MIDI player or DAW.
