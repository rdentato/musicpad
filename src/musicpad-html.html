About Musicpad

Musicpad is a textual musical notepad: you write compact notation, then generate a MIDI file. The original project was designed as a fast way to capture musical ideas from any computer, using plain text instead of a graphical sequencer.

Original project

The original Musicpad was written by Loïc Prot as a CGI Perl script, largely inspired by Jens Johansson’s polymath.cgi. The archived pages describe Musicpad as an online musical notepad with support for notes, rhythms, tracks, macros, polyrhythms, random macros, MIDI controllers, pitch bends, guitar-oriented notation, drum shortcuts, and humanization commands.

What Musicpad is good at

  • Sketching melodies quickly with note names such as C E G.
  • Writing rhythm patterns compactly with x, -, and =.
  • Layering independent tracks with |.
  • Building songs from reusable macros and repeated sections.
  • Writing drum parts with short names like iBD, iSD, and iHH.
  • Representing guitar-tab-like material with guitar mode.

The “I” project

The archived Musicpad pages included a large “I” project: a Work in Progress transcription of Meshuggah’s 21-minute song I. It showed how Musicpad could be used for long, complex, heavily structured music, with many macros, dense rhythm patterns, and multiple tracks.

This JavaScript version

This application ports the Musicpad-to-MIDI functionality to JavaScript. It runs locally in the browser: you can load a Musicpad text file, edit it, save the source, and generate a MIDI file for download. Playback is intentionally left to your own MIDI player or DAW.
